租借露營商品APP

安排露營行程不再卡卡！
露營區、活動、器材預約租借一次規劃好！

Tool Used

Project Overview

背景

戶外活動是我的喜好，露營也是其中之一。因為沒有器材，以往都是使用營區提供的，但部分品質不佳，且都是輕便帳蓬。提前規劃預約的過程也是很繁瑣。因此想設計一個擁有一整套露營服務的APP。

問題

隨著疫情趨緩，除了出國遊玩，野外露營，也變成現在流行的趨勢。但提前準備的過程卻很複雜，營區預約、活動查詢和商品出租不能在同一個平台執行。

且因為流行趨勢，非法營區增加，露營新手對露營規則的不解，導致環境的破壞。

Project Overview

解決方案

提供一個擁有一整套露營服務的APP，從查詢預約合法營區、附近活動推薦、選擇自己喜好的露營商品出租， 加入基本露營知識的補充。

讓安排露營行程方便快速、喜好的商品提升露營氣氛、同時了解基本露營知識保護環境。

台北捷運Go APP

Redesign Project - 捷護照

獲 2022北捷黑客松比賽-季軍

Taipei Metro, FIND NEW TALENTS JOIN THE HACKATHON

Project Overview

為何redesign？

台北捷運擁有每日兩百萬人流的場域，在數位化趨勢下，致力於結合新創團隊推動數位轉型。


此外，為了推廣運輸及生活事業多元服務發展策略，希望透過捷運盃黑客松競賽挖掘業界好手，針對現有台北捷運Go APP發想創新、實用的專業提案，讓APP能成為乘客不可或缺的助手。

業主需求

行動裝置是生活必須工具，希望團隊提案能提出強化現有台北捷運Go App功能、串接不同的服務，以達到提升使用體驗、串連線上與線下服務、吸引民眾下載、使用及加入會員、提供會員個人化服務等目的。

北捷規劃

目前北捷有規劃以會員系統做延伸，希望能將票卡數位化，達到綠色經濟。優化現有會員制度，新增創新功能，達到APP使用率及黏著度的提升。


隨著智慧手機的成長，遊戲化的趨勢，Z時代的崛起，手機遊戲化也歸納為創新的考量。
